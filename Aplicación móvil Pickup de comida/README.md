# Aplicación móvil Pickup de comida

## Descripción
Aplicación móvil desarrollada para la gestión de pedidos de comida, permitiendo a los usuarios visualizar opciones y realizar solicitudes de manera sencilla.

## Tecnologías utilizadas
- Kotlin
- Android Studio

## Funcionalidades
- Visualización de productos o menú
- Gestión de pedidos
- Interfaz amigable para el usuario

## Documentación del proyecto
El repositorio incluye la documentación completa del desarrollo:

- Arquitectura de software
- Backlog del proyecto
- Historias de usuario
- Diccionario de datos
- Manual de usuario
- Manual técnico
- Identidad del sistema
- Matriz de pruebas
- Plan de mantenimiento
- Presentación del proyecto
- Proyecto final

## Objetivo del proyecto
Aplicar conocimientos en desarrollo móvil mediante la creación de una aplicación funcional orientada a la gestión de pedidos.
