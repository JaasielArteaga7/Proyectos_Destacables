# Aplicación móvil Pickup de comida

## Descripción
Aplicación móvil desarrollada para la gestión de pedidos de comida, permitiendo a los usuarios visualizar opciones y realizar solicitudes de manera sencilla.

## Tecnologías utilizadas
- Kotlin
- Android Studio

## Funcionalidades
- Visualización de productos o menú
- Gestión de pedidos
- Interfaz amigable para el usuario

## Objetivo del proyecto
Aplicar conocimientos en desarrollo móvil mediante la creación de una aplicación funcional orientada a la gestión de pedidos.
