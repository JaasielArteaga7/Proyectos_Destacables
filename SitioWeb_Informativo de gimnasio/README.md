# Sitio Web Informativo de Gimnasio

## Descripción
Este proyecto consiste en el desarrollo de un sitio web informativo enfocado en la promoción de un gimnasio, mostrando sus servicios, instalaciones y contenido relevante para usuarios.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Visualización de información del gimnasio
- Presentación de servicios
- Interfaz visual atractiva

## Objetivo del proyecto
Aplicar conocimientos de desarrollo web para la creación de una página informativa funcional y visualmente atractiva.
