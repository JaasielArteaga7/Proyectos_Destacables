# Sistema Web de Venta de Productos Deportivos

## Descripción
Este proyecto consiste en el desarrollo de una aplicación web orientada a la gestión y venta de productos deportivos relacionados con el gimnasio. 

El sistema implementa una arquitectura básica de frontend y backend, permitiendo la visualización de productos y la gestión de información mediante base de datos.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- MongoDB

## Arquitectura del sistema
El proyecto se divide en dos partes principales:

- **Frontend:** Interfaz de usuario desarrollada con HTML, CSS y JavaScript.
- **Backend:** Lógica del sistema y conexión con base de datos.
- **Base de datos:** MongoDB para almacenamiento de productos.

## Funcionalidades
- Visualización de productos deportivos
- Gestión básica de productos
- Conexión a base de datos
- Interfaz web interactiva

## Objetivo del proyecto
Aplicar conocimientos de desarrollo web full stack mediante la integración de frontend, backend y base de datos en un sistema funcional.
